Final Project Ujian Akhir Semester 4

Versi server hosting silahkan masuk ke https://flexmob.my.id/

Website Flexmob adalah website platform sewa mobil dengan referensi dari traveloka rent-car dan trevo rent-car. Terdapat 3 akses dari 3 sudut pandang yang berbeda yaitu akses user, akses mitra/perusahaan dan akses admin website. Pada halaman index akan tampil tombol untuk daftar sebagai kerjasama mitra dan akses user(login terlebih dahulu, jika belum punya akun silahkan registrasi dulu)
Untuk akses sebagai Admin Website dan Admin Perusahaan Mitra silahkan masuk ke /loginadmin.php dengan "username" && "password" agar mendapatkan akses sepenuhnya mengenai traffic penjualan :
Admin website = "admin" && "1234"
Admin perusahaan A = "perusahaan a" && "perusahaana"
Admin perusahaan B = "perusahaan b" && "perusahaanb"
Admin perusahaan C = "perusahaan c" && "perusahaanc"
